# Daily .NET Challenge - March 16, 2026

## Overview
A simple C# learning project for daily challenges and practice exercises.

## Getting Started

### Prerequisites
- .NET SDK 6.0 or later
- Visual Studio or VS Code

### Setup
1. Clone the repository
2. Navigate to the project directory
3. Run `dotnet restore`
4. Run `dotnet build`

## Running the Project
```bash
dotnet run
```
